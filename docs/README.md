## Como gerar o PDF

1. Abra `parte-teorica.md` (VS Code / Typora / Pandoc / Google Docs).
2. Preencha **Autor** e **Data** no cabeçalho.
3. Exporte os diagramas Mermaid (ou redesenhe no draw.io) se a instituição exigir UML clássico.
4. Exporte para **PDF** ou **DOCX**.
5. Cole o link do vídeo na seção 9.
