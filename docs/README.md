## Como gerar o PDF

1. Preencha os placeholders de print em `prints/` (nomes sugeridos no checklist).
2. Abra `parte-teorica.md` no VS Code / Typora / Pandoc / Google Docs.
3. Exporte os diagramas Mermaid (ou redesenhe no draw.io) como imagens e substitua se a instituição exigir UML “clássico”.
4. Exporte para **PDF** ou **DOCX**.
5. Cole o link do vídeo na seção final.
